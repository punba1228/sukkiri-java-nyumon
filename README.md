リテラル
  int...30...小数点がない数字
  long...300000L...小数点がない数字で末尾がLまたはl
  double...30.5...小数点付きの数字

  boolean...true...trueまたはfalse
  char...'雅'...引用符で囲まれた文字
  String..."Java"...二重引用符で囲まれた文字列


整数リテラルの関数応用記法
  先頭0x 16進数...int a = 0x11;
  先頭0  8進数
  先頭0b 2進数 ...int c = 0b0011;

  アンダースコア記号(_)を含めることが許されているので、
  大きな数字をわかりやすく表記できる
  「2,000,000円」→「int price = 2_000_000;」

エスケープシーケンス
  \" ...二重引用符記号(")
  \' ...引用符記号(')
  \¥...円記号(¥)
  \n...改行(制御文字)

算術演算子
  +...加算（足し算）
  -...減算（引き算）
  *...乗算（掛け算）
  /...除算（割り算）
  %...剰算（割り算の余り）

文字列結合演算子
  +...文字列の結合..."こん" + "にちは" + 1 = "こんにちは1"

代入演算子
  = ...右辺を左辺に代入...a = 10 → a = 10
  +=...左辺と右辺に加算し左辺に代入...a += 2 → a = a + 2
  -=...左辺と右辺に源算し左辺に代入...a -= 2 → a = a - 2
  *=...左辺と右辺に乗算し左辺に代入...a *= 2 → a = a * 2
  /=...左辺と右辺に除算し左辺に代入...a /= 2 → a = a / 2
  %=...左辺と右辺に除算し、その余りを左辺に代入...a %= 2 → a = a % 2
  +=...(文字列)左辺の後に右辺を連結して代入...a += "風" → a = a + "風"

インクリメント・デクリメント演算子
  ++...値を１増やす...a++ → a = a + 1
  --...値を１減らす...a-- → a = a - 1

  ※ ++や--は、他の演算子と一緒に使わない

型の変換
  変数への代入時「小さい型」の値を「大きい型」に代入する際、
  代入される値が代入先の型に自動的に変換されてから代入が行われる。

キャスト演算子
  (変換先の型名)式
  元のデータの一部を失ってもデータを強制的に変化する。
  よほどの理由がない限り使わない。
